Useful commands
* Make sure you have Node.js installed!

Initialize a node application:
        npm init -y

Install dependencies:
        npm install express nodemon cors dotenv

Make sure to create a .gitignore file to ignore node_modules!
* We don't want to push node_modules to GitHub

Start server:
        node index.js

Type into search bar to see webpage:
        http://localhost:[insert port number here]

Start server:
        npm run dev
* From Stack Overflow
(https://stackoverflow.com/questions/64397005/what-are-npm-run-dev-and-npm-run-prod):
    "Used for running the specific commands for serving the project, to any server,
    to live development. In the case of a web page, you'll see your web page in the
    browser, and any change you make to the HTML code, for example, will be
    reflected immediately in the page you see in your browser."

Install Mongoose
        npm install mongoose
